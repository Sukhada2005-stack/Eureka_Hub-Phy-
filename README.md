   # Add Physics Games to Eureks Hub
    1. The Gravity Sinker
    Launch probes around a central black hole using real orbital mechanics. The trajectory appears as a glowing chalk line on a dark slate board, turning complex gravitational physics into a satisfying and intuitive "space basketball" challenge.

    2. Constellation Tracer
    A meditative "connect the dots" experience where you "sew" stars together with silk-like threads. Each successful connection plays a glassy musical note, allowing you to compose a unique ambient melody while revealing the hidden geometry of the cosmos.

    3.Spectral Slider
    Operate a vintage brass spectroscope to decode the chemical "fingerprints" of starlight. By adjusting elemental sliders to match a target star’s hue, you bathe your silent, dark laboratory in vibrant light and discover the atoms that fuel the infinite void.
  
  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
